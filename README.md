# PostScript


v0.1  
Data types  
At the moment, includes only integers.
Abilities:
1. Arithmetic operations. Supports:  
    1.1 addition  
   ```
   13 14 add
   ```
   1.2 subtraction
   ```
   100 44 sub
   ```
   1.3 multiplication  
   ```
   3 4 mul
   ```
   1.4 division 
   ```
   20 4 div
   ```
   1.5 remainder of the division
   ```
   20 14 mod
   ```
